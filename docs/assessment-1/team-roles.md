## Decision-Making
Technical architecture decisions are led by Farhaz given the role's mandate, but significant choices (e.g. tech stack, scope trade-offs) are discussed as a team before being finalised and logged as an Architecture Decision Record in the Wiki, so Isuru and Dineli retain visibility and input despite the technical skill gap.
